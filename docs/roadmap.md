# Phased Implementation Roadmap

## Phase 1

- Workspace foundation.
- Backend bootstrap.
- Frontend RTL shell.
- Environment and deployment conventions.
- Base database schema and migrations.

## Phase 2

- Auth, users, roles, permissions.
- Branch settings.
- Sequence service.
- Audit logging.

## Phase 3

- POS order lifecycle.
- Shift open and close.
- Treasury ledger.
- Receipt printing.

## Phase 4

- Ingredients, warehouses, and stock ledger.
- Recipes and standard cost.
- Auto-consumption on sales.

## Phase 5

- Vendors, purchasing, goods receipts, vendor invoices, vendor payments.
- Operating expenses.
- Setup costs and payment tracking.

## Phase 6

- Reports.
- Approval workflows.
- Hardening, testing, backups, monitoring.
